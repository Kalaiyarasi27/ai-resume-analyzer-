## Quick Start

### Prerequisites

- [Node.js](https://nodejs.org/)
- [npm](https://npmjs.com/)
- [Git](https://git-scm.com/)

### Installation

```bash
git clone https://github.com/Kalaiyarasi27/ai-resume-analyzer-.git
cd ai-resume-analyzer-
npm install
````

### Run the project

```bash
npm run dev
```

Visit: `http://localhost:5173`
